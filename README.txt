Quiz App - Project Documentation

Project Overview:
This is a simple web-based quiz application that allows users to take a multiple-choice quiz directly in their browser. The application is built using HTML, CSS, and JavaScript.

Features:
- Interactive quiz interface
- Immediate feedback for selected answers
- Score tracking
- Basic styling for a clean and user-friendly experience

Technologies Used:
- HTML
- CSS
- JavaScript

How to Run:
1. Download all files (quiz.html, quiz.css, quiz.js).
2. Open quiz.html in any modern web browser.
3. Take the quiz and view your score at the end.

Folder Structure:
- quiz.html: Main structure of the quiz interface
- quiz.css: Styling for layout and visual design
- quiz.js: Quiz logic and functionality
